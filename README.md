## 🌐 Como visualizar o projeto

### Opção 1 — Deploy com Vercel (recomendado)

1. Clique no botão **Deploy with Vercel** acima.
2. Siga os passos na plataforma Vercel (gratuito).
3. Após o deploy, você receberá uma URL no formato:  
   `https://quintal-secreto.vercel.app`

### Opção 2 — Firebase App Hosting

1. Instale o Firebase CLI: `npm install -g firebase-tools`
2. Faça login: `firebase login`
3. Inicialize o App Hosting no seu projeto Firebase:  
   `firebase apphosting:backends:create`
4. Conecte ao repositório GitHub e o deploy será automático a cada push.
5. Sua URL pública ficará disponível no console do Firebase.

## 💻 Executar Localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Configure a variável `GEMINI_API_KEY` no arquivo [.env.local](.env.local) com sua chave da API Gemini.
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Acesse no navegador: http://localhost:3000
